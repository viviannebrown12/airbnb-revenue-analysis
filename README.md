# Airbnb Revenue Analysis

## Project Goal

Analyze Airbnb listing and calendar data to identify pricing strategies that maximize host revenue.

## Business Questions

- What factors have the largest impact on Airbnb prices?
- Which amenities add the most value?
- How do prices change around holidays?
- Do higher holiday prices appear to reduce occupancy?
- Which neighborhoods generate the highest revenue?
- How should hosts adjust prices to maximize revenue?
